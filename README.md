# AI for Protein Modeling

本项目系统性整理了 **蛋白质相关 AI 模型的发展脉络（2018–至今）**，涵盖结构预测、蛋白语言模型、蛋白设计与生成等方向。  
项目以 **时间线 + 原论文 PDF + 官方 GitHub 链接 + 个人解读** 的形式进行整理，旨在构建一份清晰、可持续更新的蛋白 AI 知识库。

  - RFdiffusion / RFdiffusion2 / RFdiffusion3
  - LigandMPNN
- **新一代统一与多任务模型**
